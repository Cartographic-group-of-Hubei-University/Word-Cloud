# Word-Cloud
* 本项目涉及一种适用于LBS兴趣点多维度信息表达的词云生成方法，采用JavaScript编程语言实现中心-力导向模型的构建，然后利用[D3.js](https://d3js.org/)将表征兴趣点的标签符号化。    

* 本项目受到测绘遥感信息工程国家重点实验室开放研究基金的资助。
